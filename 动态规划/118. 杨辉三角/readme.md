### 题目 118. 杨辉三角
#### 题目描述
给定一个非负整数 `numRows`，生成杨辉三角的前 `numRows` 行。



在杨辉三角中，每个数是它左上方和右上方的数的和。

示例:

```js
输入: 5
输出:
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
]

```

示例 2:

```js
输入: numRows = 1

```
 

提示:

- 1 <= numRows <= 30
