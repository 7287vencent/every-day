### 题目 50. Pow(x, n)
#### 题目描述
实现 pow(x, n) ，即计算 x 的 n 次幂函数（即，xn）。

示例 1：

```js
输入：x = 2.00000, n = 10
输出：1024.00000
```
示例 2：

```js
输入：x = 2.10000, n = 3
输出：9.26100
```
示例 3：

```js
输入：x = 2.00000, n = -2
输出：0.25000
解释：2-2 = 1/22 = 1/4 = 0.25
 
```

提示：

`-100.0 < x < 100.0`

`-231 <= n <= 231-1`

`-104 <= xn <= 104`
