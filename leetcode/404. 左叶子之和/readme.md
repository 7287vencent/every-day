### 题目 404. 左叶子之和
#### 题目描述
计算给定二叉树的所有左叶子之和。

示例：
```js

    3
   / \
  9  20
    /  \
   15   7

在这个二叉树中，有两个左叶子，分别是 9 和 15，所以返回 24
```
