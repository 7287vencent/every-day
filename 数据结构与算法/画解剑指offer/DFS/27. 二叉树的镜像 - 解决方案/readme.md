## 题目 27. 二叉树的镜像 - 解决方案
#### 题目描述
请完成一个函数，输入一个二叉树，该函数输出它的镜像。

例如输入：
``` js
     4
   /   \
  2     7
 / \   / \
1   3 6   9
```
镜像输出：
``` js
     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
示例 1：

```
输入：root = [4,2,7,1,3,6,9]
输出：[4,7,2,9,6,3,1]
```
限制：`0 <= 节点个数 <= 1000`


### 解题思路
#### 思路
`标签：dfs`
- 递归结束条件：
- 当节点 root 为 null 时，说明已经到叶子节点了，递归结束
#### 递归过程：
- 初始化当前节点，并且赋值
- 递归原来树的右子树 mirrorTree(root.right)，并将该结果挂到当前节点的左子树上
- 递归原来树的左子树 mirrorTree(root.left)，并将该结果挂到当前节点的右子树上
- 时间复杂度：O(n)，空间复杂度：O(n)
