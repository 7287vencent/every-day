### 题目 剑指 Offer II 010. 和为 k 的子数组
#### 题目描述
给定一个整数数组和一个整数 `k` ，请找到该数组中和为 `k` 的连续子数组的个数。

 

示例 1：

```js
输入:nums = [1,1,1], k = 2
输出: 2
解释: 此题 [1,1] 与 [1,1] 为两种不同的情况
```
示例 2：

```js
输入:nums = [1,2,3], k = 3
输出: 2
```
 

提示:

- 1 <= nums.length <= 2 * 104
- -1000 <= nums[i] <= 1000
- -107 <= k <= 107
