### 题目 剑指 Offer II 044. 二叉树每层的最大值
#### 题目描述
给定一棵二叉树的根节点 root ，请找出该二叉树中每一层的最大值。

 

示例1：

```js
输入: root = [1,3,2,5,3,null,9]
输出: [1,3,9]
解释:
          1
         / \
        3   2
       / \   \  
      5   3   9 
```
示例2：

```js
输入: root = [1,2,3]
输出: [1,3]
解释:
          1
         / \
        2   3
示例3：

输入: root = [1]
输出: [1]
```
示例4：

```js
输入: root = [1,null,2]
输出: [1,2]
解释:      
           1 
            \
             2     
```
示例5：

```js
输入: root = []
输出: []
```
 

提示：

- 二叉树的节点个数的范围是 [0,104]
- -231 <= Node.val <= 231 - 1
