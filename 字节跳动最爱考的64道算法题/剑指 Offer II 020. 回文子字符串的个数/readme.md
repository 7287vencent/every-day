### 题目 剑指 Offer II 020. 回文子字符串的个数
#### 题目描述
给定一个字符串 `s` ，请计算这个字符串中有多少个回文子字符串。

具有不同开始位置或结束位置的子串，即使是由相同的字符组成，也会被视作不同的子串。

 

示例 1：

```js
输入：s = "abc"
输出：3
解释：三个回文子串: "a", "b", "c"
```
示例 2：

```js
输入：s = "aaa"
输出：6
解释：6个回文子串: "a", "a", "a", "aa", "aa", "aaa"
```
 

提示：

- `1 <= s.length <= 1000`
- `s 由小写英文字母组成`
